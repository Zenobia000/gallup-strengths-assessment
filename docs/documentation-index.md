## 快速開始

### 使用者
1. 閱讀 [使用者手冊](./user-manual-zh.md)
2. 按照評估說明進行
3. 下載並檢視您的結果

### 管理員
1. 查看 [維護手冊](./maintenance-manual.md)
2. 按照安裝程序進行
3. 設定監控和備份系統
4. 實施安全最佳實踐

## 系統需求

### 終端使用者需求
- 現代網路瀏覽器 (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
- 穩定的網路連線
- 啟用 JavaScript
- PDF 檢視器用於報告下載

### 系統管理員需求
- Python 3.9+
- PostgreSQL 或 SQLite
- Redis 伺服器
- Linux/Unix 環境 (建議)
- 基本命令列知識

*如有技術問題，請聯繫系統管理員。*
